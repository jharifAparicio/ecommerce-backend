## tecnogias utilizadas
-> Express(typescript)

-> Claudinary(almacenamineto de imagen y otros)

-> turso para la base de datos

## 1. Instalar Dependecias
    
```console
$ npm install
```

## 2. compiar TypScript ( si es nesesario)

```console
$ npx tsc
```

## 3. levantar el servidor en modo desarrollo

```console
$ npm run dev
```
